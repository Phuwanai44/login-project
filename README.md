# Fullstack Login System

Login system built with Vue, TypeScript, Go (Gin), and MongoDB.

## Tech Stack

Frontend
- Vue
- TypeScript
- Bootstrap

Backend
- Go (Gin)
- MongoDB
- JWT Authentication

## Features

- User Register
- User Login
- JWT Authentication
- Role-based access (Admin / User)

## API Endpoints

POST /register
POST /login
GET /profile
